[Overwolf](https://www.overwolf.com/) provides middleware for tracking game metrics (score, player kills, etc.) in real-time for multiple triple-A PC game titles. Apps created for the Overwolf platform provide in-game overlays directly in the game (similar to Steam). I managed all technical and design responsibilities for the Jackpot Rising Overwolf app. This app collected metrics utilized in our competitive tournaments. The application itself was a desktop application that utilized web-based technology on the frontend (think Electron or Tauri), implemented Vue.js as the frontend framework, and communicated directly with the Overwolf API.
